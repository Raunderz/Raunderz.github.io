What is Rta

Rta is an open development ecosystem that puts an AI coding agent in your terminal, on your phone, or in your IDE. It routes through a shared backend that handles authentication, AI provider fallback, rate limiting, and telemetry — so you get a consistent experience everywhere.

The CLI is the core: 24 built-in tools (file ops, git, search, LSP, MCP), 13 LLM provider integrations, streaming TUI, session persistence, and a minimal REPL mode. Everything routes through the Rta backend which auto-falls back across Groq, Cerebras, SambaNova, OpenRouter, and Gemini.

How it works

  • CLI sends your prompt + tools to the Rta backend
  • Backend authenticates, checks rate limits, routes to the best available provider
  • Provider (Groq, Cerebras, Gemini, OpenRouter, SambaNova) generates a response
  • CLI executes any tool calls (read files, run commands, search the web) and loops until done
  • Session is persisted as JSONL — resume anytime with full context

Tech

  • CLI — Python 3.12+, Textual TUI, aiohttp, pydantic
  • Backend — FastAPI, Supabase, Stripe, SlowAPI
  • Container Service — Go, Docker, WebSocket, Cloudflare Tunnel
  • Mobile — Expo / React Native 0.83, CodeMirror 6, isomorphic-git
  • Desktop — C/Lua (Lite XL fork), Odin agent
  • Website — Preact, Vite, wouter, marked

Status

CLI and backend are production-ready. Container service is deployed. Mobile app, desktop IDE, and website are functional with ongoing development. Everything is MIT licensed.
